> <h3> Getting start : </h3>
 <br/>

<div>

1. <b>Clone the project</b>

```bash
  $ git clone https://github.com/gabriel-durr/rick-and-morty-api.git && cd api-rick-and-morty
```

2. <b>Install dependencies</b>

```bash
  $ npm i
```

3. <b>Start the application</b>

```bash
  $ npm run dev
```
